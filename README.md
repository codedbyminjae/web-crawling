---

## 📚 About the Course

This repository contains weekly assignments and examples from the **"데이터 크롤링" (Data Crawling)** course.

Students explore real-world web scraping techniques using Python libraries such as:

- `requests`
- `BeautifulSoup`
- `Selenium`
- `pandas`
- `lxml` and more.
